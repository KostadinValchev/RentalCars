# RentalCars

## This is a simple web application for rental cars.

### Application

* Register users – User name, Email,First name, Last name, Password, Birth date.
* Login/Logout users – User name, Password.
* Agency Panel - Agency can manage their cars.
* Manage Booking - Users can see their booking history.

### Usage

#### Roles:

* Admin
* Agency

#### Rights:

##### Admin:

*Add and delete Cities.

##### Agency:

* Add, delete and update cars

##### Logged in users:

* User can book car.
* User can see their history and last  reservation details.
